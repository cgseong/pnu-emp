GitHub 저장소 정보 입력

현재 상황:
- 로컬 폴더가 Git으로 초기화됨
- 이제 GitHub의 원격 저장소와 연결해야 함

필요한 정보:
1. GitHub 사용자명: ?
2. GitHub 저장소명: pnu-emp (인 것 같음)
3. GitHub 저장소 URL: https://github.com/[사용자명]/pnu-emp.git

다음 정보를 제공하면 정확한 명령어를 알려드리겠습니다:

[ ] GitHub 사용자명은 무엇인가요?
    (예: sw40904, john-doe 등)

[ ] 현재 GitHub 저장소 URL은 정확히 무엇인가요?
    (https://github.com/... 형태)

[ ] 또는 Streamlit Cloud의 앱 설정에서 연결된 GitHub 저장소를 확인할 수 있습니다
    (Streamlit Cloud → 앱 → "Manage app" → 저장소 URL 확인)
